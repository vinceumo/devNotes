`hsl()` defines color using the HSL(A) model:

- `H`: Hue
- `S`: Saturation
- `L`: Lightness
- `A`: Alpha (optional)

![HSL color wheel]({{ site.baseurl }}/dist/img/hsl.jpg)

```
color: hsl(`hue from 0 to 360`, `saturation from 0% to 100%`, `lightness from 0% to 100%` [`Alpha from 0% to1 100%`])
```
