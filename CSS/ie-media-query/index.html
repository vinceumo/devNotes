Hack - CSS target only IE11 and IE10 (media query)

Aug 08, 2018

Small hack, to apply CSS to only Internet Explorer 10 and 11, using media query:

@media all and (-ms-high-contrast: none), (-ms-high-contrast: active) {
  /* Your styles */
}
